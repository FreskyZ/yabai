直播相关接口
============

> 大致描述了接入一个直播平台需要的接口

## 需求

+ **获取直播推流**
+ 直播相关信息
    + 直播状态
    + 封面
    + 标题
    + 房间uid
    + 头像
+ 弹幕/sc
+ 直播间添加
    + 获取分区热榜
    + 导入关注
